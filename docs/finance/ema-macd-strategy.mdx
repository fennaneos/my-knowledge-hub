# EMA–MACD Bias Strategy

**Idea.** Use the medium-term trend (``EMA(12)`` vs ``EMA(26)``) for *bias* and time entries with the **MACD histogram** zero-cross; gate with **RSI(14)** to avoid extremes.

## Rules

**Bias**

- Bullish if \( \operatorname{EMA}_{12} > \operatorname{EMA}_{26} \)  
- Bearish if \( \operatorname{EMA}_{12} < \operatorname{EMA}_{26} \)

**Trigger**

- Long when \( \operatorname{Hist} = \operatorname{MACD} - \operatorname{Signal} \) crosses **above 0** and bias is **bullish**.  
- Short when it crosses **below 0** and bias is **bearish**.

**RSI filter**

- Skip long if \( \operatorname{RSI}_{14} > 70 \)  
- Skip short if \( \operatorname{RSI}_{14} < 30 \)

**Exit**

- Reverse on opposite signal (baseline). Extensions: stop/TP, time-based exit.

## Formulas

- \( \operatorname{EMA}_k = \alpha P_t + (1-\alpha)\operatorname{EMA}_{k-1}, \quad \alpha=\frac{2}{k+1} \)
- \( \operatorname{MACD} = \operatorname{EMA}_{12} - \operatorname{EMA}_{26} \)
- \( \operatorname{Signal} = \operatorname{EMA}_9(\operatorname{MACD}) \)
- \( \operatorname{Hist} = \operatorname{MACD} - \operatorname{Signal} \)

## Pseudocode

```python
# identifiers inside a fenced code block are safe
ema12 = EMA(close, 12)
ema26 = EMA(close, 26)
macd, signal, hist = MACD(close, 12, 26, 9)
r = RSI(close, 14)

for t in range(1, N):
    biasUp = ema12[t] > ema26[t]
    biasDn = ema12[t] < ema26[t]
    crossUp = hist[t-1] <= 0 and hist[t] > 0
    crossDn = hist[t-1] >= 0 and hist[t] < 0
    longOK  = biasUp and crossUp and r[t] <= 70
    shortOK = biasDn and crossDn and r[t] >= 30
